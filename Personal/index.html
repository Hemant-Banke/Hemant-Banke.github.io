Usefulness of Ensemble Methods for creating Xg models

July 2022
Open Report

  • Expected goals (or xG) measures the quality of a chance (between 0 and 1) by calculating the likelihood that it will be scored from a particular position on the pitch during a particular phase of play. This value is based on several factors from before the shot was taken.
  • Project aimed at developing an xG model. This can be achieved by fitting a binary classification model and treating the probability of classification as xG.
  • The goals were to perform feature selection, feature engineering, compare different models to create best xG model ; facing a challenge of properly training and tuning these models. Used 'caret' package in R to ease training and tuning.
  • Training Setup :
    1. k-fold repeated cross validation : for tuning the hyper parameters (used default grid search for possible values considered while tuning).
    2. Model Selection : Choose the hyper parameters resulting in the highest ROC AUC metric.
    3. ROSE Sub-sampling : The training data has an obvious heavy imbalance, hence used ROSE sub-sampling under repeated-cv (training folds are sub-sampled for tuning).
  • Comparison of Models :
    Trained following Non-ensembled models : Logistic Model (glm), Penalized Logistic model (glmnet), Decision Trees (rpart)
    Bagged Models : Random Forest (rf)
    Boosted Models : AdaBoost (ada.m1 and real ada), XGBoost (xg), Boosted Logistic
    Stacked Models : Compared performance of (glmnet, rpart) against (glmnet, knn) as base models, and glm against gbm (Gradieant Boost machine) as meta models.
  • Feature Engineered discrete variables (DefendersInCone) and a binary variable (shot.open_goal) into their continuous counterpart using kernel density estimation (assuming each defender is a 2D normally distributed random variable with mean as the defender's current position).

Regression Analysis of Diabetes data

December 2021
Open Report

  • Aimed to observe the dependence of Insulin in blood plasma on relative weight, fasting glucose level, random glucose level and steady state glucose level for Normal, Chemically Diabetic and Overt Diabetic patients.
  • The difference in predicted insulin levels in the three cases gave an insight on how insulin helps in transferring glucose to cells and how this process is affected during Diabetes.
  • Performed Multiple Linear Regression on the data after dealing with the following challenges :
    1. Non normal distribution of response variable : fixed by performing Box-Cox transformation
    2. Multicollinearity : fixed by methods like Ridge regression, feature selection by performing Lasso Regression
    3. Influential Points : Removed after being found through measures like Cook's Distance, DFBETA, DFFITS, Covratio.
    4. Non-linearity : Observed presence of non linearity through Added variable plot and partial residual plot for variable 'sspg'. Using log transform fixed the issue.
  • Performed Model selection using All possible Regression with AIC as performance criteria

Open Report

The project aimed at simulating various kinds of Markov chain and doing relevant study :
